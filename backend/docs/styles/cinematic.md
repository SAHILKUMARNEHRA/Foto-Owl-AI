# Cinematic Style Guide

Use a clear three-act emotional arc even in short edits.

Prioritize emotionally legible images with strong subject focus and dynamic lighting.

Captions should be sparse, intentional, and poetic rather than descriptive.

Favor slow push-ins, parallax drift, gentle zooms, and motivated fades.

Color direction should be cohesive across scenes so the edit feels authored instead of shuffled.
